# Restaurant Online Booking Website

This is a simple restaurant online booking website with a login/signup feature and a form for user login. The website provides a beautifully designed header and footer, along with a background image and information about the restaurant.

## Features

- User login and sign-up buttons
- User login form
- Background image showcasing the restaurant
- Header and footer sections

## Technologies Used

- HTML
- CSS
